Introductie

In deze python cursus wordt dieper ingegaan op het werken met modules, classes en ai-toepassingen. Python is de op gebied van AI een van de belangrijkste programmeertalen. Er zijn al veel modules beschikbaar waardoor je kunt werken met populaire LLM's.
In deze cursus gaan we dieper in op systemen en applicaties die gebruik maken van AI en LLM's.

Doel van de cursus

  • Leren werken met AI, classes en modules met python
  • Leren werken met python als taal om AI applicaties te bouwen.
  • Python verdieping op data-analyse
  • Python als taal voor IoT-oplossingen

Voorwaarden om aan deze cursus te beginnen

  • Je hebt basiskennis van Python
  • Je hebt enige ervaring met prompten en het werken met AI chatsystemen.
  • Je hebt ervaring met Git en Github. Je kunt applicatie veilig opslaan en beheren met deze systemen.
